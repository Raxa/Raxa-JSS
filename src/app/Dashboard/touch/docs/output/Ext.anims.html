Class Ext.anims

Package:Ext
Defined In:Anim.js
Class:anims
Extends:Object

Defines different types of animations. flip, cube, wipe animations do not work on Android.

Please refer to Ext.Anim on how to use animations.



This class is a singleton and cannot be created directly.
