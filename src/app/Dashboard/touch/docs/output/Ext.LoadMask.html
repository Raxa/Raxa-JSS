Class Ext.LoadMask

Package:Ext
Defined In:LoadMask.js
Class:LoadMask
Extends:Object
A simple utility class for generically masking elements while loading data. If the store config option is specified, the masking will be automatically synchronized with the store's loading process and the mask element will be cached for reuse.

Example usage:

// Basic mask:
var myMask = new Ext.LoadMask(Ext.getBody(), {msg:"Please wait..."});
myMask.show();
